# Polegion Documentation

This directory contains all project documentation, examples, and SQL scripts.

## Directory Structure

### `/examples/`
Contains example chapter implementations that serve as templates for creating new chapters:
- `EXAMPLE_CASTLE2_CHAPTER1.tsx` - Example implementation for Castle 2, Chapter 1
- `EXAMPLE_CASTLE5_CHAPTER1.tsx` - Example implementation for Castle 5, Chapter 1

These files are safe references for developers creating new chapter content and can be kept for future development.

### `/sql/`
Contains database schema, migrations, and seed data:
- `DATABASE_COMPLETE_SCHEMA.sql` - Complete database schema for the application
- `CREATE_ASSESSMENT_TABLES.sql` - Assessment system table creation
- `INSERT_ASSESSMENT_QUESTIONS.sql` - Assessment question seed data
- `CASTLE_0_6_ASSESSMENT_SEED.sql` - Castle 0-6 assessment seed data
- `CASTLE_0_6_MANUAL_INSERT.sql` - Manual insert scripts for castle data
- `FIXED_CASTLE_XP_TRACKING.sql` - XP tracking system fixes

### Root Documentation Files
- `PROJECT_DOCUMENTATION.md` - Main project documentation
- `ASSESSMENT_CASTLES_SETUP_SUMMARY.md` - Assessment system setup guide
- `AUDIO_IMPLEMENTATION_SUMMARY.md` - Audio system documentation
- `CAROUSEL_UPDATE.md` - Castle carousel update notes
- `CASTLE_PROGRESSION_GUIDE.md` - Castle progression system guide
- `CHAPTER_TEMPLATE_GUIDE.md` - Guide for creating new chapters
- `CHAPTER2_3_MIGRATION_GUIDE.md` - Migration guide for chapters 2-3
- `COMPLETE_CHAPTER_REFERENCE.md` - Complete chapter reference
- `HOW_TO_USE_CHAPTER_TEMPLATE.md` - Chapter template usage
- `IMAGE_REQUIREMENTS_CASTLE_0_TO_6.md` - Image asset requirements
- `LINE_AND_ANGLE_IMPLEMENTATION.md` - Line and angle feature docs
- `PHASE_2_COMPLETE.md` - Phase 2 completion notes
- `PROPERTIES_PANEL_IMPLEMENTATION.md` - Properties panel documentation
- `SIMPLE_CHAPTER_GUIDE.md` - Simplified chapter creation guide

## Usage Notes

- **Example files** should be kept as reference templates for future development
- **SQL files** may be needed for database setup, migrations, or data restoration
- All files in this directory are safe to keep and serve as project knowledge base
